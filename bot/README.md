# Telegram Task Management Bot

Бот для управления задачами через Telegram с поддержкой голосовых сообщений и AI-обработкой.

## Стек

- Python 3.11+
- aiogram 3 — фреймворк для Telegram-ботов
- Supabase — база данных (PostgreSQL)
- OpenAI (через Replit AI) — транскрипция голоса и извлечение задач

## Возможности

- Создание задач из текстовых и голосовых сообщений
- Автоматическое определение заголовка, описания, дедлайна и категории через AI
- Управление задачами: просмотр, перемещение по колонкам, смена категории

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Регистрация пользователя |
| `/tasks` | Все активные задачи (сгруппированы по колонкам) |
| `/today` | Задачи из колонки "сегодня" |
| `/done <номер>` | Отметить задачу выполненной |
| `/move <номер> <колонка>` | Переместить задачу в другую колонку |
| `/tag <номер> <категория>` | Изменить категорию задачи |
| `/categories` | Список категорий |
| `/columns` | Список колонок доски |

## Структура проекта

```
bot/
├── main.py              # Точка входа, запуск polling
├── config.py            # Конфигурация из переменных окружения
├── database.py          # Supabase клиент и запросы к БД
├── handlers/
│   ├── commands.py      # Обработчики команд /start, /tasks и т.д.
│   └── messages.py      # Обработчики текста и голоса
├── services/
│   ├── ai_service.py    # Транскрипция и извлечение задач через AI
│   └── task_service.py  # Бизнес-логика задач
├── requirements.txt
└── .env.example
```

## Переменные окружения

| Переменная | Описание |
|-----------|----------|
| `TELEGRAM_BOT_TOKEN` | Токен бота от @BotFather |
| `SUPABASE_URL` | URL проекта Supabase |
| `SUPABASE_SERVICE_ROLE_KEY` | Service Role ключ Supabase |
| `AI_INTEGRATIONS_OPENAI_BASE_URL` | Replit AI прокси URL (устанавливается автоматически) |
| `AI_INTEGRATIONS_OPENAI_API_KEY` | Replit AI ключ (устанавливается автоматически) |

## Таблицы в базе данных

Бот использует существующие таблицы (не создаёт новые):
- `tg_users` — пользователи
- `tg_tasks` — задачи
- `tg_categories` — категории
- `tg_board_columns` — колонки Kanban-доски

## Запуск

```bash
cd bot
pip install -r requirements.txt
python main.py
```
